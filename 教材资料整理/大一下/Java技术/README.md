## 考试说明

平时成绩包括：作业（课后习题，甚至有答案），实验（最后一题选做，需要验收），课堂测试/期中考试（不难），期末考试。

闭卷考试，用纸笔答题。有选择题，填空题，简答题，判断结果题，编程题等。整体难度适中，可以重点复习一下类，对象，继承，接口等相关内容，后续多线程/IO/GUI部分考的很少，基本均为送分题。
